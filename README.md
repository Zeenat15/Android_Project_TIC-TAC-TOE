# Android_Project_TIC-TAC-TOE

Tic Tac Toe is a classic two-player game where players take turns marking spaces in a 3x3 grid with X and O. The objective is to be the first player to get three of their marks in a row, either horizontally, vertically, or diagonally. This Android project, implemented in Java, allows two players to play against each other on a single device. The game features a simple and intuitive user interface, making it easy to enjoy this timeless game on your mobile device.
